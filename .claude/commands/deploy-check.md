# /deploy-check

本番デプロイ前のチェックリストを実行してください。

## チェック項目

1. **ビルド確認**: `npm run build` が成功するか
2. **環境変数チェック**: `.env.example` と `.env.local` の差分を確認
3. **型チェック**: `npm run check` がエラーゼロ
4. **Lint**: `npm run lint` がエラーゼロ
5. **テスト**: `npm run test` が全パス
6. **セキュリティ**:
   - `.env.local` がgitignoreに含まれているか
   - ハードコードされたシークレットがないか（APIキー、パスワード等）
   - `NEXT_PUBLIC_` 環境変数に機密情報が含まれていないか
7. **DB**: 未適用のマイグレーションがないか
8. **依存関係**: `npm audit` で critical/high の脆弱性がないか

## 出力形式

チェックリスト形式で結果を表示:

```
デプロイ前チェック:
  [✅] ビルド成功
  [✅] 環境変数: 差分なし
  [❌] セキュリティ: .env.local が gitignore に未登録
  ...

  結果: X/Y 項目パス → デプロイ可能 / 要修正
```
