# /check-all

プロジェクト全体の品質チェックを実行してください。

## 実行内容

以下の3つを順番に実行し、結果をまとめて報告:

1. **Lint**: `npm run lint`（Biome）
2. **型チェック**: `npm run check`（tsc --noEmit）
3. **テスト**: `npm run test`（Vitest）

## 出力形式

結果を以下の形式でサマリ表示:

```
品質チェック結果:
  Lint:     ✅ PASS / ❌ FAIL (エラー数)
  型チェック: ✅ PASS / ❌ FAIL (エラー数)
  テスト:    ✅ PASS (X件) / ❌ FAIL (X/Y件失敗)
```

## エラー時

- 各チェックでエラーがあれば詳細を表示
- 修正が必要な場合は具体的な修正箇所を提案
- 全て通過した場合は「コミット可能」と表示
