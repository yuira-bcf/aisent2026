この画面について

AIが配合計算に使用する香料の基礎データ（L1 物理層）を登録する画面です。
各香料の物理的特性を定義することで、AIが適切なノートバランスの配合を生成できるようになります。

入力項目の説明

  • 香料名（日本語/英語） — ユーザー画面に表示される名称。日本語名は必須。
  • ノートタイプ — TOP（最初に香る）/ MIDDLE（中心の香り）/ LAST（余韻として残る）から選択。
  • カテゴリ — 香料の系統分類。AIのグルーピングに使用。
  • 揮発性（0.00〜1.00） — 0に近いほど長く持続し、1に近いほどすぐ揮発する。
  • 香りの強さ（0.00〜1.00） — 0に近いほどほのかで、1に近いほど強く香る。
  • 重さ — エアリー（軽い）/ バランス / リッチ（重い）。全体の重量感設計に使用。
  • 温度感 — クール / ニュートラル / ウォーム。フレグランスマップのx軸座標に対応。
  • メモ — 香料の特徴。AI生成のストーリーテキストにも反映。
0.00（持続性高い）1.00（すぐ揮発）
0.00（ほのか）1.00（非常に強い）

この画面について

入力手順

  3. 備考 に具体的な理由を記入（任意）。
  4. 「禁止ペアを登録する」で保存。
