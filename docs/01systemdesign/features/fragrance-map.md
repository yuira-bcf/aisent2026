## 1. 概要

ユーザーの調合結果を2次元マップ上にプロットし、有名フレグランス25種との位置関係と類似度を可視化する機能。配合ベクトル（26香料の比率）から座標計算とコサイン類似度計算を行う。

---

## 2. 座標軸定義

### 2.1 軸の意味

```
                    フローラル (+1)
                        │
                        │
   フレッシュ × フローラル │ ウォーム × フローラル
                        │
  フレッシュ (-1) ───────┼──────── ウォーム (+1)
                        │
   フレッシュ × ウッディ  │ ウォーム × ウッディ
                        │
                        │
                    ウッディ (-1)
```

| 軸 | 負の方向 (-1) | 正の方向 (+1) | 対応L1属性 |
|----|-------------|-------------|-----------|
| X軸 | フレッシュ (Fresh) | ウォーム (Warm) | 温度感 (1-6) |
| Y軸 | ウッディ (Woody) | フローラル (Floral) | 重さ (1-5) / カテゴリ (1-4) |

座標範囲: x, y ともに `[-1.0, +1.0]`

## 4. 計算ロジック

実装: `apps/web/lib/fragrance-map/similarity.ts`

### 4.1 calculatePosition: マップ座標算出

配合ベクトル（香料名 → 比率%）から2次元座標を算出する。

```typescript
export function calculatePosition(recipe: Record<string, number>): MapPosition
```

**アルゴリズム: 加重平均**

```
入力: recipe = { 香料名: 比率(%) }

x = Σ(FLAVOR_COORDINATES[香料].x * ratio) / Σ(ratio)
y = Σ(FLAVOR_COORDINATES[香料].y * ratio) / Σ(ratio)

出力: { x, y }  （range: [-1.0, +1.0]）
```

**計算例:**

```
配合: ライム 20%, ジャスミン 30%, バニラ 10%

x = ((-0.95 * 20) + (0.10 * 30) + (0.75 * 10)) / (20 + 30 + 10)
  = (-19.0 + 3.0 + 7.5) / 60
  = -0.142

y = ((-0.20 * 20) + (0.85 * 30) + (0.10 * 10)) / (20 + 30 + 10)
  = (-4.0 + 25.5 + 1.0) / 60
  = 0.375

結果: { x: -0.142, y: 0.375 }
→ フレッシュ寄り、フローラル寄り
```

### 4.2 cosineSimilarity: コサイン類似度

26香料を次元とする配合ベクトル間のコサイン類似度を計算する。

```typescript
export function cosineSimilarity(
  vecA: Record<string, number>,
  vecB: Record<string, number>,
): number
```

**アルゴリズム:**

```
全26香料をキーとする次元空間で:

similarity = (A . B) / (||A|| * ||B||)

         Σ(a_i * b_i)
  = ─────────────────────────
    √(Σ(a_i^2)) * √(Σ(b_i^2))

出力: 0.0（完全に異なる）〜 1.0（完全一致）
```

- 存在しない香料の値は 0 として扱う
- 両方のベクトルが零ベクトルの場合は 0 を返す

### 4.3 findSimilarFragrances: Top N検索

ユーザーの配合ベクトルに最も近いリファレンスフレグランスを返す。

```typescript
export function findSimilarFragrances(
  userVector: Record<string, number>,
  topN = 3,
): SimilarResult[]
```

**処理フロー:**

```
1. 全25リファレンスフレグランスのvectorとユーザーのvectorでコサイン類似度を計算
2. 類似度の降順でソート
3. 上位N件（デフォルト3件）を返却
```

**戻り値:**

```typescript
type SimilarResult = {
  fragrance: ReferenceFragrance;  // リファレンスフレグランスの全情報
  similarity: number;             // 0.0〜1.0
};
```

## 6. 型定義

実装: `apps/web/lib/fragrance-map/types.ts`

```typescript
// 香料座標
export type FlavorCoordinate = {
  x: number;   // -1.0〜+1.0
  y: number;   // -1.0〜+1.0
};

// リファレンスフレグランス
export type ReferenceFragrance = {
  id: string;
  name: string;
  nameJa: string;
  brand: string;
  category: string;
  gender: 'women' | 'men' | 'unisex';
  vector: Record<string, number>;       // 26香料への近似配合（合計1.0）
  position: { x: number; y: number };   // vectorから自動計算
};

// 類似度計算結果
export type SimilarResult = {
  fragrance: ReferenceFragrance;
  similarity: number;   // 0.0〜1.0
};

// マップ上の座標
export type MapPosition = {
  x: number;
  y: number;
};
```
