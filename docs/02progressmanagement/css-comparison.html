残留差異 (対応不要)

意図的な技術差異またはモック固有の要素。修正不要。

技術差異 ページ構造

モック: .screen-wrap (position:fixed SPA風) / 実装: Next.js App Router (ファイルベースルーティング)。技術スタックの違いによる意図的な差異。

技術差異 CSS手法

モック: CSS変数 + カスタムクラス / 実装: Tailwindユーティリティ主体。ビジュアル結果は同一。

モック固有 追加ブレークポイント・トークン

--color-accent #be185d, --color-text-tertiary #999999 (モック定義のみ)

実装固有 デスクトップナビリンク

hidden md:flex でデスクトップ時にナビバー内にリンク表示。モックはドロワーのみ。
